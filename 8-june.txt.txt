DSA:

-> garbage collector - clears unused objects in heap memory
-> heap memory - used for storing objects

Time complexity - time taken to evaluate or execute a program
O(n) - worst case scenario
O(1) - direct access without loops
O(n) - single loop
O(n^2) - two nested loops
O(log n) - higher than 1 and less than n
Coefficient omitted for complexity


Space complexity - extra space used
